We can apply CSS by two method internal and external

Internal CSS is written within the <style> tag in the <head> section of the HTML document. It applies styles to elements on that specific page.

External CSS is written in a separate file with a .css extension and linked to the HTML document using the <link> tag. It allows for consistent styling across multiple pages.
